itemComplianceValidator
--------------

Requisitos de ejecución: 

Este paquete requiere contar con java 1.8 o superior instalada en la máquina en la que se va a ejecutar


--------------

Este paquete contiene un archivo itemComplianceValidator.bat (Windows) o itemComplianceValidator.sh (Linux/MacOS). 

--------------
Para ejecutarlo en MAC/Linux ejecute en un terminal el comando: 
sudo chmod 700 itemComplianceValidator.sh
y luego ejecute: 
./itemComplianceValidator.sh

--------------
Para ejecutarlo en windows sólamente ejecute: 
itemComplianceValidator.bat

Para los dos casos siga las instrucciones que aparecen en pantalla. 


--------------
Sitio base del proyecto: https://github.com/robertohern83/itemComplianceValidator.git
